# LaunchCode — AI Intelligence for Builders

Daily briefings on foundation models, video AI, AI agents, and funding rounds — built for the global AI-native startup ecosystem.

## Tech Stack

- **Frontend**: React + Tailwind CSS (dark theme, Replicate-inspired)
- **Backend**: FastAPI + PostgreSQL
- **Fonts**: Inter + JetBrains Mono
- **Accent**: Violet `#7c3aed`

## Coverage

AI Models · Video AI · AI Agents · Generative AI · Funding · Research · Infrastructure · Computer Vision

## Dev

```bash
cd frontend && npm install && npm start
cd backend && pip install -r requirements.txt && python server.py
```
